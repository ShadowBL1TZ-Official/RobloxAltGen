## Installation

1. Clone this repository to your local machine.
2. Install the necessary dependencies by running `pip install -r requirements.txt`.
3. Run the main script using `python main.py`.
4. Edit main.py to the directory of the repository
5. Make sure everything from requirements.txt, If something is not installed you can manually install it by copying the name and putting this command `pip install (paste here and remove parentacies)`

## Usage

Once you have installed My Awesome Software, you can use it as follows:

1. Open a terminal or command prompt.
2. Navigate to the directory where you cloned the repository.
3. Run the main script using `python main.py`.
4. Follow the on-screen instructions to use the software.

When you run it, It will continuesly generate to accounts.txt until you exit out of command prompt or stop the script
